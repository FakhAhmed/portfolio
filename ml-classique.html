Prédiction Business (ML)

Ce projet est actuellement en cours de réalisation.

À Propos du projet :

Ce projet "Flash" se concentre sur les fondamentaux de la Data Science appliquée aux problématiques métier (détection de fraude bancaire ou prédiction de désabonnement client). L'objectif est de démontrer une maîtrise parfaite de l'ingénierie des caractéristiques (Feature Engineering) et de la préparation de données structurées.


Objectifs et Fonctionnalités Prévues :
  • Analyse Exploratoire (EDA) : Identification des corrélations, gestion des valeurs aberrantes et déséquilibres de classes.
  • Feature Engineering Propre : Création de nouvelles variables pertinentes pour le modèle mathématique.
  • Entraînement Modèle : Utilisation d'algorithmes de boosting à base d'arbres (XGBoost) très prisés dans l'industrie financière.
  • Interprétabilité : Explicabilité des prédictions du modèle pour le métier (ex: Feature Importance, SHAP values).

Stack Technique Envisagée :
  • Scikit-Learn
  • XGBoost
  • Pandas / NumPy
  • Matplotlib / Seaborn
Les liens vers le code source (Notebook) et les résultats seront disponibles une fois le projet finalisé.
