Business Prediction (ML)

This project is currently in progress.

About the project:

This "Flash" project focuses on the fundamentals of Data Science applied to business challenges (e.g., bank fraud detection or customer churn prediction). The objective is to demonstrate perfect mastery of feature engineering and structured data preparation.


Planned Objectives and Features:
  • Exploratory Data Analysis (EDA): Correlation identification, handling outliers, and managing class imbalances.
  • Clean Feature Engineering: Creating new relevant variables for mathematical modeling.
  • Model Training: Using tree-based boosting algorithms (XGBoost), highly valued in the financial industry.
  • Interpretability: Explaining model predictions for business stakeholders (e.g., Feature Importance, SHAP values).

Planned Tech Stack:
  • Scikit-Learn
  • XGBoost
  • Pandas / NumPy
  • Matplotlib / Seaborn
