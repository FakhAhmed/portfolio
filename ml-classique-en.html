MLOps Prediction System

About the project:

This project illustrates the transition from theoretical Data Science to software industrialization (MLOps). It features an end-to-end architecture, from training an XGBoost model to its production deployment on Google Cloud Platform.


Why this project? This PoC demonstrates the ability to transform an isolated mathematical model into an enterprise microservice: resilient to data errors, deployed via automated pipelines (GitOps), and secured by strict IAM strategies. This is the very essence of modern Data & AI Engineering.


Key Features:
  • Code/Model Decoupling (Dynamic Fetching): The model is no longer "hardcoded" in the Docker image. The API dynamically downloads the .bst file at startup from a secure Cloud Storage Bucket.
  • Model Governance (Vertex AI): Integration of Google's Model Registry. The API queries the official catalog to always fetch and serve the certified production version.
  • Native CI/CD (GitOps): Fully automated deployment via Google Cloud Build. Each git push triggers the Docker build and Serverless deployment without human intervention.
  • "Least Privilege" Security (IAM): The API runs under a dedicated Service Account, granted only strict read permissions, ensuring total isolation in the event of a breach.
  • Resilient API: FastAPI microservice with Pydantic validation and a "lifejacket" implementation (Data Imputation) to dynamically align incoming requests with the features required by the model.

Project Structure:
  • /api/: Source code of the REST API (FastAPI). Handles JSON validation, Vertex AI querying, and inference.
  • cloudbuild.yaml: The brain behind the automation. Orchestrates the CI/CD pipeline steps (Build, Push, Deploy).
  • /notebooks/: Research lab containing Exploratory Data Analysis (EDA) and initial model training.
  • Dockerfile: Containerization recipe ensuring absolute application portability.

Technical Architecture:
  • XGBoost / Scikit-Learn
  • FastAPI / Uvicorn
  • Python / Pandas
  • GCP Cloud Run
  • Google Cloud Build
  • Google Cloud Storage
  • Vertex AI Model Registry
  • Cloud IAM (Security)
  • Docker (Artifact Registry)
