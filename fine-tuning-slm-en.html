Fine-Tuning Factory (SLM)

This project is currently being architected.

About the project:

Designing a complete MLOps pipeline dedicated to re-training Small Language Models (SLM). The goal is to take a lightweight, high-performance model (e.g., Llama-3 8B or Phi-4) and specialize it for a specific domain (law or medicine), while ensuring rigorous experiment tracking via MLflow.


Planned Objectives and Features:
  • PEFT / LoRA: Efficient Parameter Training (Low-Rank Adaptation) to adapt the model without requiring expensive GPU clusters.
  • Speed Optimization: Integration of the Unsloth library to drastically accelerate fine-tuning times.
  • MLOps & Traceability: Using MLflow to compare hyperparameters, training loss, and manage the model registry.
  • Data Versioning: Tracking training datasets via DVC (Data Version Control).
  • Real-time Serving: Deploying the specialized model behind a production-ready FastAPI API.

Planned Tech Stack:
  • Llama-3 8B / Phi-4
  • LoRA / Unsloth
  • MLflow
  • DVC
  • FastAPI
  • Hugging Face
The code repository and MLflow metrics will be available soon.
