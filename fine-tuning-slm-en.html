Fine-Tuning Factory (SLM)

About the project:

An end-to-end Machine Learning architecture, from data versioning to the fine-tuning of a Small Language Model (SLM) and its web deployment. This project demonstrates the implementation of a complete MLOps pipeline to specialize a powerful language model (Llama-3 8B) on financial sentiment analysis tasks, orchestrated across the Google Cloud and Hugging Face ecosystems.


Why this project? This PoC illustrates the real-world engineering complexity surrounding Large Language Models (LLMOps). It proves the ability to go beyond a simple Python script to build a true software "factory": capable of managing versioned data, optimizing the training of billions of parameters, and rigorously tracking experiments on the Cloud.


Key Features:
  • PEFT / LoRA Fine-Tuning: Specializing the Meta Llama-3 foundation model without retraining all weights. Utilization of 4-bit quantization (Unsloth) for drastic VRAM optimization (training on a T4 GPU).
  • Data Versioning (DVC): Financial data is treated as code. Its history is strictly versioned via Data Version Control (DVC) with secure remote storage on Google Cloud Storage.
  • Experiment Tracking (MLflow): Deployment of a proprietary MLflow server on Google Cloud Run (Serverless). Every metric (Loss) is logged live in the cloud.
  • Interactive Inference (Hugging Face): The fine-tuned model is publicly hosted and queried via a seamless web interface developed with Gradio, proving the model's "Serving" capability.

Project Structure:
  • /data/: Storage space for training datasets, tracked and versioned by DVC.
  • /notebooks/: Central training laboratory executing Unsloth fine-tuning and pushing metrics to the remote MLflow server.
  • /src/: Source code for the web interface (Gradio). Dynamically loads the base model and the LoRA adapter.
  • dvc.yaml: The brain of the data pipeline to GCP.

Tech Stack & Architecture:
  • Llama-3 8B / PyTorch
  • LoRA / Unsloth
  • DVC (Data Versioning)
  • MLflow (Tracking)
  • GCP Cloud Run
  • GCP Cloud Storage
  • Hugging Face Hub
  • Gradio (Web UI)
  • Python / TRL
