Statistique sur les accidents

A Propos du projet:

Notre application permet de répondre à de nombreuses questions autour des accidents et le type de luminosité. Celle-ci est connectée au serveur lakartxela avec le driver ODBC afin d’effectuer les requêtes de façon dynamique. L’utilisateur utilisant notre application aura le choix selon plusieurs critères (5 au total) :

  • Sur le nombre d'accidents.
  • Sur le nombre d'accidents graves.
  • Sur le degré de gravité.
  • Sur les intempéries principales.
  • Sur les causes principales.
