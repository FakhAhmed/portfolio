Usine de Fine-Tuning (SLM)

Ce projet est actuellement en cours de réalisation.

À Propos du projet :

Concevoir une usine MLOps complète dédiée au ré-entraînement de Petits Modèles de Langage (Small Language Models). L'objectif est de prendre un modèle léger performant (type Llama-3 8B ou Phi-4) et de le spécialiser sur un domaine métier précis (droit ou médecine), tout en garantissant un suivi rigoureux des expériences via MLflow.


Objectifs et Fonctionnalités Prévues :
  • PEFT / LoRA : Entraînement paramétrique efficace (Low-Rank Adaptation) pour adapter le modèle sans exiger de clusters GPU hors de prix.
  • Optimisation de la vitesse : Intégration de la librairie Unsloth pour accélérer drastiquement les temps de fine-tuning.
  • Traçabilité MLOps (Expérimentations) : Utilisation de MLflow pour comparer les hyperparamètres, la perte d'entraînement (loss) et gérer le registre des modèles.
  • Versionning des Données : Suivi des datasets d'entraînement via DVC (Data Version Control).
  • Serving Temps Réel : Déploiement du modèle spécialisé derrière une API FastAPI prête pour la production.

Stack Technique Envisagée :
  • Llama-3 8B / Phi-4
  • LoRA / Unsloth
  • MLflow
  • DVC
  • FastAPI
  • Hugging Face
